# Contributing

Dziękujemy za zainteresowanie projektem. Jeśli chcesz wnieść wkład:

- Otwórz issue opisujące propozycję lub błąd.
- Zrób forka i wyślij pull request zawierający opis zmian.
- Zachowaj czytelność commitów i dodaj krótki opis w PR.

Proste zmiany (dokumentacja, drobne poprawki) są mile widziane.
